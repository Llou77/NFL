How It Works

A stacked ML ensemble trained on the last 3 NFL seasons (816 regular season + 39 playoff games). Every Tuesday during the season, GitHub Actions automatically fetches updated stats, retrains the model on a rolling window, and publishes new predictions.

Each prediction outputs a score for both teams — from which spread, total, and moneyline predictions are derived automatically.

Model Architecture

Layer 1A
Ridge Regression
Linear baseline with L2 regularisation.
Layer 1B
XGBoost
Gradient boosting, MAE objective, 500 estimators.
Layer 1C
LightGBM
Fast leaf-wise gradient boosting.
Layer 2
Dual-Head NN
PyTorch 256→128→64, separate home/away heads. Multi-target loss.
Layer 3
Meta-Learner
Ridge blends all sub-model outputs via out-of-fold predictions.

Confidence Score

Model Agreement (40%)

Std-dev of sub-model predictions. High disagreement → lower confidence.

Feature Completeness (30%)

Fraction of expected features available. Missing QB status or injury data penalised.

H2H Data Quality (15%)

Sample size of historical matchup games. Division games score near 1.0.

Injury Freshness (15%)

<24h before kickoff = 1.0. >7 days out = 0.4.

Sliding Window

816 regular season + 39 playoff games (3 full seasons). Season weights auto-tuned by Bayesian optimisation (Optuna) every pre-season. GitHub Actions runs the full retrain every Tuesday at 10:00 UTC.
