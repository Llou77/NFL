How the Model Works

Every Tuesday during the NFL season, an automated pipeline fetches updated statistics from nflverse, retrains a stacked ML ensemble on the last 3 seasons (816 regular season + 39 playoff games), and generates predictions for every upcoming game.

The model predicts a final score for both teams — from which spread, total, and moneyline predictions are automatically derived. Season weights are optimised by Bayesian search (Optuna) every pre-season rather than fixed manually.

Model Architecture

Layer 1A
Ridge Regression
Linear baseline with L2 regularisation.
Layer 1B
XGBoost
500 trees, MAE objective, handles non-linear interactions.
Layer 1C
LightGBM
Fast leaf-wise gradient boosting alternative.
Layer 2
Dual-Head Neural Net
PyTorch 256→128→64. Separate home/away heads. Multi-target loss simultaneously optimises score, spread, and total.
Layer 3
Meta-Learner
Ridge regression that learns the optimal blend of all sub-model outputs via out-of-fold predictions.

Matchup Gap Features

For each game: offensive EPA of Team A minus defensive EPA allowed by Team B — for passing, rushing, 3rd down, and red zone. This directly quantifies the strength mismatch in each phase.

Head-to-Head History

Historical matchup records with tiered lookback: 3 seasons for division rivals (6 meetings available), 6 seasons for conference opponents, 10 seasons for cross-conference. H2H weight is reduced proportionally if teams have undergone major roster changes since those games.

Confidence Score

Every prediction carries a 0–100% confidence score. This does not mean probability of winning — it measures how much the model trusts its own prediction given data quality. Edge indicators are hidden when confidence is below 50% (WEAK) to avoid misleading signals from data gaps.

Model Agreement — 40%
How closely Ridge, XGBoost, LightGBM, and NN predictions agree. High std-dev between models = lower confidence.
Feature Completeness — 30%
Fraction of expected features available. Missing QB status or injury data penalised more heavily.
H2H Data Quality — 15%
Sample size of historical head-to-head games. Division rivals score near 1.0; cross-conference games with 0 recent meetings score 0.2.
Injury Freshness — 15%
<24h before kickoff = 1.0. >7 days out = 0.4. Predictions made early in the week have lower confidence.
